## 발표용 보충 사실 (질문 대비, 슬라이드엔 안 넣어도 됨)
- 학습 데이터: 우리 합성 1000 scene → scene 단위 800/100/100 split (S5와 동일).
- 입력 모드 `zv` = 정규화 depth + valid mask 2채널. 배경 NaN→valid=False.
- num_classes 27, num_queries 100, decoder 6층·8헤드, hidden 256, lr 1e-4, batch 4, AMP off.
- CAD codebook (27, 256)은 frozen — detector가 cosine 유사도로 cad_id 예측.
- 평가는 Hungarian 매칭 후 matched 객체 기준 정확도/IoU (eval_test.json 원본 첨부).
- class acc와 CAD acc가 ~80%로 비슷 = 두 경로(라벨 분류 / 형상 codebook)가 일관. 둘 다 depth만으로 27종.
- box IoU(0.682)가 mask IoU(0.855)보다 낮음 = 길쭉/얇은 부품에서 bbox가 마스크보다 불리(발표 시 솔직하게).
