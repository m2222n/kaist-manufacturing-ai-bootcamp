## 프로젝트 목표

산업 현장에서 출력한 3D 프린터 부품을, **다양한 환경에서도 실제 카메라가 정확하게 인식**하도록 만드는 비전 AI 모델을 개발한다.

- 단순 평가셋 지표(mAP 등)는 중간 신호일 뿐, **실제 카메라로 부품을 정확히 인식**하는 것이 종착점이다.
- 데이터셋 구축 → 전처리 → 모델링 → 검증까지 한 사이클을 6주 안에 완주한다.
- 산출물(데이터·모델·코드)은 발표 이후 실제 산업 적용까지 이어가는 것을 염두에 둔다.

> 세부 접근 방식(데이터셋 구성·인식/복원 알고리즘 등)은 주차별 미팅을 거치며 확정해 나가는 단계이며, 확정되는 대로 본 README에 반영한다.

## 6주 진행 계획

| 주차 | 작업 |
|------|------|
| 1주차 | 데이터셋 구축 및 구축 계획 |
